# GetFiles

GetFiles is a simple PHP class to return a list of files in a directory.

[Read The Documentation](https://digi-brains.github.io/get-files/)

### Installation

Via composer.json

```
{
    "require": {
        "digi-brains/get-filesd" : "<a specific release or 'dev-master'>"
    }
}
```
